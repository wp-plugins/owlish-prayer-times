Displays prayer times from islamicfinder.org - please note, this plugin needs the xml-url created at islamicfinder.org

== Description ==

Displays prayer times in a highly customizable way. No iframe used, you can change the look of the prayer times to fit your page. Basic CSS included, so it will adapt to the style of your theme (maybe some modifications needed).

== Installation ==

1. Upload `owlish-prayertimes.zip` to the `/wp-content/plugins/` directory or use the "Add new" dialogue on the WordPress backend.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the widget via the appearance menu in your wordpress admin interface.
4. Configure the widget as explained in the tutorial: <a href="http:/codingowl.net/freebies/owlish-prayer-times-wordpress-plugin" target="_blank">to the tutorial</a>

== Frequently Asked Questions ==

= Can this plugin work with individual prayer times? =

Yes, you can use the highly customized prayer times from islamicfinder.org. On the website you can setup the calculation method, location, juristic method and much more for your prayer times. Please refer to the <a href="http:/codingowl.net/freebies/owlish-prayer-times-wordpress-plugin" target="_blank">tutorial</a>.

= Can this plugin work without the XML-URL from islamicfinder.org? =

No, this plugin works only with the data retrieved from the islamicfinder.org Website.
